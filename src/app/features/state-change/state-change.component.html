Basic State Change Component

Key Points

  • Wild card state: * represents any state including void state. When there are more than 2 states and we want to define same animation transition among all of those states then use *. Eg * <=> * means any-to-any state transition.
  • Void state: use the void state to configure transitions for an element that is entering or leaving a page.
    Eg. * => void OR :leave (Element is leaving)
    Eg. void => * OR :enter (Element is entering regardless of its previous state).
